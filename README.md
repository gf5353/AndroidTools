# 安卓开发工具

### 批处理工具

- 查看APK文件信息
> 将APK直接拖至bat文件上即可，信息保存在bat所在目录下的txt文件里

- 自动安装APK
> 将APK直接拖至bat文件上即可

- 读取手机信息存至文件
> 双击bat文件即可

- 快速手机截屏
> 双击bat文件即可

- 导出Logcat日志
> 双击bat文件即可。可以按需要修改adb logcat命令，比如monkey前可以用adb logcat -c 清空缓存，后再执行此bat时，日志文件则不会太多无用的日志。
